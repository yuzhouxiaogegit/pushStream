# youtube 推流脚本

### 环境centos 6/7/8
##### 1、首先需手动要安装ffmpeg
```sh
wget https://raw.githubusercontent.com/yuzhouxiaogegit/pushStream/main/ffmpeg_install.sh && chmod +x ffmpeg_install.sh
./ffmpeg_install.sh
```
##### 2、推流脚本安装
```sh
wget https://raw.githubusercontent.com/yuzhouxiaogegit/pushStream/main/push_video.sh && chmod +x push_video.sh
./push_video.sh
```
